pyArch
======

Tools (simulator, assembler) for a new developed cpu architecture.

## Directory structure
* arch  -> simulator (reference implementation)
* asm   -> assembler
* doc   -> documentation

## Warning
Everything here is experimental! Meaning: expect the worse ;-)
